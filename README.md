# 🏥 MedicaCare - Sistem Informasi Klinik & Apotek

MedicaCare adalah aplikasi web berbasis **Laravel + React (Inertia.js)** yang digunakan untuk mengelola sistem informasi klinik dan apotek, termasuk data pasien, jadwal dokter, rekam medis, resep digital, dan stok obat dalam satu platform modern.

---

## 🚀 Tech Stack

### Backend

- Laravel (via laravel-vite-plugin)
- Inertia.js (server-side integration)
- Axios (HTTP request)

### Frontend

- React 19
- Vite (build tool)
- Tailwind CSS 4
- React Icons
- Alpine.js (untuk kebutuhan interaktif ringan)

### Build Tools

- Vite
- PostCSS
- Autoprefixer
- Tailwind Vite Plugin
